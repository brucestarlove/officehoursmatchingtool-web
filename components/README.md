# Components Directory

This directory follows atomic design principles:

## Structure

- **`ui/`** - Base UI components (atoms)
  - CF-branded components: `button-cf.tsx`, `card-cf.tsx`, `badge-cf.tsx`
  - shadcn/ui components (to be added)

- **`auth/`** - Authentication components (molecules/organisms)
  - Login forms, signup forms, etc.

- **`mentor/`** - Mentor-related components
  - Mentor cards, profiles, availability calendars

- **`sessions/`** - Session-related components
  - Booking flows, session cards, session details

- **`match/`** - Matching components
  - Search bars, filters, mentor grids

- **`feedback/`** - Feedback components
  - Feedback forms, rating stars

- **`dashboard/`** - Dashboard components
  - Widgets, stats cards, suggestions

- **`admin/`** - Admin components
  - Analytics dashboards, data tables

- **`shared/`** - Shared components
  - Reusable components used across features
